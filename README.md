# DelTool (批量删除工具)

## 工具说明

*  因为出更新包时，要去解压再删除特定的文件夹，在包比较多的时候，比较费时，所以有了此工具。

## 使用要求

+ 必须安装WinRar软件

+ 使用的压缩文件解压后生成一个和压缩文件名字一样的文件夹（不能直接解压在root目录下）
